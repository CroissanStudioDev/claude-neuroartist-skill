# Platform Sizes

## VK

| Format | Size | image_size |
|--------|------|------------|
| Post | 1080x1080 | `square` |
| Story | 1080x1920 | `portrait_16_9` |
| Cover | 1920x768 | custom |
| Avatar | 400x400 | `square` |
| Product | 1000x1000 | `square` |

## Telegram

| Format | Size | image_size |
|--------|------|------------|
| Sticker | 512x512 | `square` |
| Avatar | 640x640 | `square` |
| Link preview | 1280x640 | custom |
| Story | 1080x1920 | `portrait_16_9` |

## Instagram

| Format | Size | image_size |
|--------|------|------------|
| Post (square) | 1080x1080 | `square` |
| Post (portrait) | 1080x1350 | `portrait_4_3` |
| Story/Reels | 1080x1920 | `portrait_16_9` |
| Profile | 320x320 | `square` |

**Tip:** 4:5 portrait gets more feed space

## YouTube

| Format | Size | image_size |
|--------|------|------------|
| Thumbnail | 1280x720 | `landscape_16_9` |
| Shorts | 1080x1920 | `portrait_16_9` |
| Video | 1920x1080 | `landscape_16_9` |
| Banner | 2560x1440 | custom |
